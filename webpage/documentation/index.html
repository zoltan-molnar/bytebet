Trumpet Premium HTML Template

Intorduction

Trumpet is a clean, responsive, neatly designed premium HTML template, built by with developers in mind. It was built with the purpose of expediting the build process of a new website, and a lot of thought had been put into the structure of each section of the website.

Getting Started

Local Deployment

Requirements

The best way to deploy Trumpet is to build it on a local machine first, and then copy the neccessary files over to a server. In order to do that you will need Node and NPM installed on your computer. You can check to see if it is installed by typing in your console:

npm -v
5.3.0

If you do not get a version number it means you do not have NPM installed. You can follow the installation instructions here.

You will also need to have Webpack installed on your computer globally. Once you have npm installed, you can install webpack by typing in in your console: 

npm install -g webpack

Installation

When you complete your purchase you are prompted to download a trumpet.zip file. Unzip it and move the folder to your preffered loaction. Open your console and navigate to the directory of the template like so: 

cd path/to/trumpet

Install the npm packages by typing the next command:

npm install

Once the packages have finished installing you will need to compile the JS and SCSS files. You can easily do this by typing: 

webpack

You will be able to see that two new files have been added to the public directory: bundle.js and styles.min.css.

In order to view the website simply open on the file trumpet/public/index.html to open it in your favorite browser.

Editing

Sass

Trumpet was built in order to house different types of content. In order to maintain the tempalte maticulous design, make sure that you copy the classes of each element you subtitute. There may be some cases where you will need to tweak the css of the website. Trumpet uses Sass to preprocess the css files of the website. If you are unfamiliar with Sass, you can find more about it here. In order for the browser to read the new styles the SCSS files need to be compiled into regular CSS. You can easily achieve this by navigating in your console to the directory of Trumpet and typing: 

webpack --watch

Leave your console open and continue editing the SCSS files. Webpack will automatically recompile the files whenever you make a change.

Projects Section

The projects section was built with the use of a plugin called Isotope. Each button has a property called "data-filter" that sets the class this button will present. You can find out more in the isotope website.

Testimonials Section

The projects section was built with the use of a plugin called Slick. When adding more testimonials, make sure to copy the classes exactly for the plugin to work.

Setting up the Map

In order to set up the map in the template you need to do two things:

  1. Enter your API key which you get from google by following these instructions.
  2. Adding the longtitude and latitude values of the address you want shown on the map.

API Key

Once you have the API key, go to pubic/index.html and scroll all the way to the bottom of the page. Find the link to the Google Maps script and enter your API key like so: 

<script type="text/javascript" src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY_GOES_HERE"></script>

Map Address

In order to get the longtitude and latitude values of your address you can use various online generators like this one. After you have the values go to js/script.js. At the top of the file you will find a place to enter your values: lat for latitude and lng for longtitude like so: 

var office = {lat: 40.748541, lng: -73.985758};

Deployment

Deployment of Trumpet is extremely easy. Once you have built the website locally, set your details, and edited the content to your liking, simply copy the public directory's contents to your website's folder on your hosting service via FTP.
